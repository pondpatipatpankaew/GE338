# การวิเคราะห์ความเหมาะสมของพื้นที่จำลองดาวอังคารในภาคตะวันออกเฉียงเหนือ

## บทนำ

การค้นหาพื้นที่จำลองดาวอังคาร (Mars Analog Site) เป็นแนวทางที่ใช้ในการศึกษาพื้นที่บนโลกที่มีลักษณะคล้ายคลึงกับพื้นผิวดาวอังคาร เพื่อใช้เป็นพื้นที่ทดสอบและศึกษาด้านรีโมตเซนซิงและภูมิสารสนเทศ ภาคตะวันออกเฉียงเหนือของประเทศไทยมีลักษณะภูมิประเทศแบบที่ราบสูงและดินสีแดงจากเหล็กออกไซด์ รวมถึงมีช่วงฤดูแล้งที่ยาว จึงมีศักยภาพในการใช้เป็นพื้นที่จำลองดาวอังคาร งานวิจัยนี้มุ่งวิเคราะห์ความเหมาะสมของพื้นที่ในภาคตะวันออกเฉียงเหนือโดยใช้ข้อมูลจาก Google Earth Engine เพื่อระบุพื้นที่ที่มีลักษณะใกล้เคียงกับพื้นผิวดาวอังคารมากที่สุด

---

## 1. Problem Statement

โครงการนี้มีวัตถุประสงค์เพื่อวิเคราะห์ความเหมาะสมของพื้นที่จำลองดาวอังคารในภาคตะวันออกเฉียงเหนือของประเทศไทย ซึ่งมีลักษณะภูมิประเทศแบบที่ราบสูงและดินสีแดงที่อาจมีความคล้ายคลึงกับพื้นผิวดาวอังคาร การศึกษานี้ใช้ข้อมูลรีโมตเซนซิงจาก Google Earth Engine โดยคำนวณดัชนี Iron Oxide Index เพื่อระบุพื้นที่ดินสีแดง ใช้ NDVI เพื่อกรองพื้นที่ที่มีพืชพรรณหนาแน่น และใช้ค่าความลาดชันจากข้อมูล DEM เพื่อเลือกพื้นที่ที่มีลักษณะราบเรียบ การวิเคราะห์ดังกล่าวมีเป้าหมายเพื่อคัดเลือกพื้นที่ที่มีศักยภาพสำหรับใช้เป็นพื้นที่จำลองดาวอังคาร ซึ่งสามารถสนับสนุนการศึกษาด้านรีโมตเซนซิง การวิเคราะห์เชิงพื้นที่ และการทดลองภาคสนามในบริบทประเทศไทย ทั้งนี้ การศึกษาใช้ข้อมูลที่มีอยู่ใน Google Earth Engine ทำให้สามารถดำเนินการได้ภายในระยะเวลาที่กำหนดโดยไม่ต้องเก็บข้อมูลภาคสนามเพิ่มเติม

## 3. Data Inventory

### ภาพดาวเทียมที่ใช้

| รายการ       | รายละเอียด                     |
| ------------ | ------------------------------ |
| ดาวเทียม     | Landsat 8 Collection 2 Level 2 |
| ช่วงเวลา     | มีนาคม 2023 – เมษายน 2025      |
| พื้นที่ศึกษา | ภาคตะวันออกเฉียงเหนือ          |

เลือกช่วงฤดูแล้งเนื่องจากพืชพรรณมีความหนาแน่นต่ำ ทำให้เห็นพื้นผิวดินชัดเจนและลดความสับสนจากพื้นที่เกษตร

### Dataset ที่ใช้

| Dataset        | การใช้งาน          |
| -------------- | ------------------ |
| Landsat 8      | คำนวณดัชนีพื้นผิว  |
| NASADEM        | คำนวณ slope        |
| ESA WorldCover | mask ป่า เมือง น้ำ |

### ดัชนีที่ใช้

| ดัชนี      | สูตร                                | วัตถุประสงค์        |
| ---------- | ----------------------------------- | ------------------- |
| Iron Oxide | B4 / B2                             | ระบุพื้นที่ดินสีแดง |
| NDVI       | (B5-B4)/(B5+B4)                     | กรองพืชพรรณ         |
| BSI        | ((B6+B4)-(B5+B2))/((B6+B4)+(B5+B2)) | ยืนยันดินเปล่า      |
| Slope      | DEM                                 | เลือกพื้นที่ราบ     |

## 4. Methodology Flow

การวิเคราะห์เริ่มจากการกำหนดพื้นที่ศึกษาภาคตะวันออกเฉียงเหนือของประเทศไทย จากนั้นดึงข้อมูลภาพถ่ายดาวเทียม Landsat 8 ในช่วงฤดูแล้งและทำการกรองเมฆ ก่อนนำภาพที่ได้มาคำนวณดัชนี Iron Oxide Index, NDVI และ BSI เพื่อวิเคราะห์ลักษณะพื้นผิวดินและความหนาแน่นของพืชพรรณ พร้อมทั้งคำนวณค่าความลาดชันจากข้อมูล DEM ต่อมาทำการ Mask พื้นที่ป่า เมือง และแหล่งน้ำออกโดยใช้ข้อมูล land cover แล้วทำการปรับค่าข้อมูลให้อยู่ในช่วงเดียวกัน จากนั้นรวมตัวแปรทั้งหมดด้วยวิธี Weighted Overlay เพื่อสร้างแผนที่ความเหมาะสม และแสดงผลเป็น Suitability Map พร้อมระบุพื้นที่ที่มีศักยภาพสูงสุด

## หากต้องลด Scope ลง 50%

หากจำเป็นต้องลดขอบเขตของโครงการลง จะลดพื้นที่ศึกษาจากทั้งภาคตะวันออกเฉียงเหนือเหลือเฉพาะบางจังหวัดที่มีศักยภาพสูง และลดตัวแปรเสริมออก โดยยังคงใช้ตัวแปรหลัก ได้แก่ Iron Oxide Index, NDVI และ Slope ในการวิเคราะห์ เพื่อให้สามารถดำเนินการได้ภายในระยะเวลาที่กำหนด ขณะที่ยังคงรักษาวัตถุประสงค์หลักของโครงการในการค้นหาพื้นที่จำลองดาวอังคารไว้
